### Enable form state restoration
Form state restoration is a browser feature that preserves user input after page reloads, navigation, or when the browser suggests autocomplete options.
This means users won't lose their form entries if they accidentally refresh the page or navigate away and return.
In native HTML forms, this behavior is enabled by default.

If you want to enable this behavior with Nectary inputs, you can add the `data-form-state-restore="true"` attribute to the form element.

```html
<form data-form-state-restore="true">
```
