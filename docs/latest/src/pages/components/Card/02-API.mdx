# API

`sinch-card-v2`

## Slots

### media

For images and illustrations. Shown at the top of the card.

### title

Can be used with the ready made `sinch-card-v2-title` or your own component. Shown at the top of the card below the media slot.

### content

Text content. Shown at the middle of the card.

### footer

Shown at the bottom of the card.
