# Examples

Accordion is a list of vertically stacked headers that allow users to show and hide information. It's mainly used to help users to process big amount of information without cluttering the UI.

## Complex

In this complex implementation of an accordion component the optional text and icons can be used to provide additional context or information about the content within each accordion item, or to help users quickly identify and navigate to specific sections of content.

The disabled state allows designers to selectively disable certain accordion items, so that they cannot be expanded or contracted by the user. This can be useful for highlighting certain sections of content, or for disabling certain items that are not relevant in a particular context.

## Multiple

The ability to expand multiple accordion itens independently allows users to view multiple pieces of content within the accordion at the same time, providing greater flexibility and control.

## Status

The inclusion of a status indicator together with the title can provide additional context and information, and can help users quickly identify and navigate to the information they are looking for.

The status indicator can be used to indicate the current status or state of the accordion content, such as whether it has been completed, is in progress, or requires further action.
