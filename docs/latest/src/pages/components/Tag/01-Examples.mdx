# Examples

A tag  works only as a visual element is a user interface element that is used to display information or status in a compact and visually appealing way. It is typically represented as a small label or badge, and is often used to highlight specific pieces of information or to indicate the status of an item.

Tag components that work only as visual elements do not have any interactive features or functionality. They are purely decorative and are used to draw the user's attention to specific information or to provide context for other content on the page.

The colors in that are allowed in the tags are the same `color` value as [Color Menu](?path=/components/colormenu/examples).
