# Examples

Rich Textarea is a Slack-like WYSIWYG editor, which serializes content into Markdown

## Composition

See [Rich Textarea compositions page](?path=/compositions/richtextarea/examples) for full-featured WYSIWYG editor.

## Hotkeys

| Key | Action |
| ---- | ---- |
| `Ctrl` / `Cmd` + `I` | toggle selection *italic* |
| `Ctrl` / `Cmd` + `B` | toggle selection **bold** |
| `Ctrl` / `Cmd` + `Shift` + `X` | toggle selection ~strikethrough~ |
| `Ctrl` / `Cmd` + `Shift` + `C` | toggle selection `codetag` |
| `Tab` | indent list item |
| `Shift` + `Tab` | outdent list item |

## Chips

Chips are rendered as chips using the `{{chipName}}` syntax. Use the `chipResolver` property to customize each chip's icon and color based on its name. The resolver is set via the component ref and does not affect the markdown output.

In this example, we also handle typing `@` to open a popover with available chips to insert at the current caret position.
