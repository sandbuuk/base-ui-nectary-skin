# Examples

The action menu is a user interface element that presents a list of actions that a user can take in a specific context. It is typically represented as a list of options that are triggered by the user clicking on the action menu item. Action menus are often used to provide users with quick access to a range of common actions without cluttering the interface with too many buttons or controls.

The specific options available in the action menu will depend on the context in which it is being used. For example, in Nectary Action Menu is a building block for high-level compositions like Dropdown, Search with autocomplete or Popover.
