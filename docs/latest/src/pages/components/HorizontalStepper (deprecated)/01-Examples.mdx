# Examples

A horizontal stepper component is a user interface element that allows users to navigate through a series of steps or stages in a process. The horizontal stepper component is a useful tool for guiding users through complex or multi-step processes. It can be used in a variety of contexts, such as onboarding flows or form wizards, to help users complete tasks more efficiently and with fewer errors. By providing a clear and intuitive navigation structure, the horizontal stepper component can help users understand and complete the steps required to complete a task.

Horizontal Stepper is an *informative* progress stepper.

## Composition

* `index` controls the currently active item
* it's only possible to change item `status` on already passed steps
