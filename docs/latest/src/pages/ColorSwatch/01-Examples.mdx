# Color Swatch

## Examples

Color Swatch is a building block for components like [Color Menu](?path=/components/colormenu).

### All

The color swatch component with a list of all possible colors can be useful for designers who need to use a wide range of colors in their designs, or who want to provide users with a comprehensive set of color options. The color swatch can be customized to fit the needs of the application, and can help users choose the right colors for their designs.
