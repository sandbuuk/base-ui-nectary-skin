# Chip

## Examples

Chip is an interactive [Tag](?path=/components/tag) that can be deleted by the user. They are a compact user interface element that is used to represent a piece of information or a discrete entity.

Chips are often used to help users navigate and access information more efficiently, by allowing them to quickly filter or organize content. Chips are typically displayed in a group, and can be customized with different colors, icons, and other design elements.

Works with the same `color` value as [Color Menu](?path=/components/colormenu).
