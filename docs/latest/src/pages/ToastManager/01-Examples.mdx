# Toast Manager

## Examples

Toast Manager displays short messages about app processes in the screen. It is mostly used to provide feedback in a user action recently taken by the user. Toast are always dismissable and have a timeout in the aplication. They're meant to be noticed without disrupting a user's experience or requiring an action to be taken.

### Complex

A complete example that generates toasts with random text, type, optional close button and persistent mode. Click a lot to see all the use cases.

> Note that having real unique IDs is crucial for the manager to work properly, just an array index won't work as expected as a `key` in React.

Toast Manager respects `(prefers-reduced-motion: reduce)` [CSS media feature](https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-reduced-motion) such as "System Preferences → Accessebility → Display → Reduce motion" in macOS.
