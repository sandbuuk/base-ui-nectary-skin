## Examples

A dropdown is a user interface element that allows the user to select from a list of options. The list of options in the dropdown will typically be organized in a logical or meaningful way, and will include visual indicators or labels to help the user understand what each option represents. Dropdowns are often used to provide the user with quick access to a range of options or settings without cluttering the interface with multiple separate buttons or controls.In nectary, a dropdown can have multiple kinds of composition.

Dropdown is a composition of a target component, [Popover](?path=/components/popover) and [Action Menu](?path=/components/actionmenu).
