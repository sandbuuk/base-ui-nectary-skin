# Examples

Alert is a banner located at the top at the UI that presents pressing and high-signal messages. Their goal is to communicate with users in a prominent way to drive their attention about a service disruption or critical information, that is or will impact the use of the product in a global way.

## Type

### Informative

Informative Alert conveys non-critical information to the user.

<Example Component={TypeInfoExample}/>

### Warning

Warning Alert conveys critical information to the user that may disrupt the user experience in a near future.

<Example Component={TypeWarningExample}/>

### Error

Error Alert conveys information that is disrupting the user experience at the moment to the user and that might require an immediate action to solve it.

<Example Component={TypeErrorExample}/>

The close button dismisses the Alert.
