# Examples

A dialog component is a user interface element that allows users to view and interact with content in a modal window. It is typically used to display additional information or options, or to prompt users for input or decisions.

The dialog component consists of a container that overlays the main content of the application, and includes a title, text, and one or more buttons or other interactive elements. The dialog can be dismissed by the user, either by clicking a close button or by clicking outside of the dialog container.

They can be used to display detailed information or instructions, to present users with options or choices, or to prompt them for input or feedback. Dialog components are often used in conjunction with other user interface elements, such as buttons or forms, to create a more seamless and intuitive user experience.

## Note on dialog close events

The `onClose` callback receives a detail string indicating how the dialog was closed:
- `'close'` - Close button was clicked
- `'escape'` - Escape key was pressed
- `'backdrop'` - User clicked outside the dialog

```tsx
<Dialog
  open={isOpen}
  onClose={(detail) => {
    if (detail === 'escape') {
      // User pressed Escape
    }
    setIsOpen(false)
  }}
/>
```
