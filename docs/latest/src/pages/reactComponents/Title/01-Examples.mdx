# Examples

The Title component allows you to add a title or heading to the user interface. It typically consists of a large, bold font that is used to draw attention and convey the main purpose or topic of the content it is paired with. The title component is an important tool for creating hierarchy and structure within the UI, helping users to quickly understand the content and navigate your user interface.

Whether you are adding a title to a page, a section, or a block of content, the Title component is an essential part of Nectary.
