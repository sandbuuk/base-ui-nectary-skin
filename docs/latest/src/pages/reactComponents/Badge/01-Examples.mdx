# Examples

The Badge component is a visual small status indicator. It is always associated with another element in the view. It can contain a counter or not. It is often used to indicate unread messages or notifications, or to show the number of items in a user's shopping cart or to-do list. Badge attaches itself to a target component such as Avatar.

## Size

The Badge component comes in different sizes. Use the size correctly so it balances well with the target component.
