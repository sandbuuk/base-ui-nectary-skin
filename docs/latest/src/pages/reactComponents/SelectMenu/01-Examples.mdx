## Custom Search

You can make the search appear with less options by using `searchable`. You can also control the search behaviour by using `onSearchChange` with `searchValue`.

Note that the "debounce" functionality is disabled when `searchValue` is implemented manually.

## Custom Options

A select option can also display arbitrary content. A `customContent` prop may be provided to render elements inside the select option. If `customContent` is provided, the `icon` prop will still be rendered if available, but the `text` property will not be rendered.

The `text` property is still required as it serves other purposes such as search.
