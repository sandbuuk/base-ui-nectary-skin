## Title Variants

The `CardTitle` component supports an optional `icon` prop to display icons alongside titles. The icon placement can be controlled with the `orientation` prop:

* `orientation="horizontal"` (default): Icon appears beside the title
* `orientation="vertical"`: Icon appears above the title

<Example Component={TitlesExample}/>

## Clickable

A clickable card provides visual feedback through hover and active state styles. The card automatically becomes clickable when an `onClick` handler is provided. You can also explicitly set the `clickable` prop.
