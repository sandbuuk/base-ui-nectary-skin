# Examples

The Button component is a React wrapper for user interactions.

## Variant

### Primary

A primary button is used to initiate the most important action on a page or screen.

<Example Component={TypePrimaryExample}/>

### Secondary

A secondary button is always used in combination with a primary button for less important actions.

<Example Component={TypeSecondaryExample}/>

### CTA Primary

A call to action (CTA) button is used to encourage the user to take a specific action.

<Example Component={TypeCtaPrimaryExample}/>

### CTA Secondary

The secondary CTA button is always used in combination with a primary CTA button.

<Example Component={TypeCtaSecondaryExample}/>

### Subtle Primary

<Example Component={TypeSubtlePrimaryExample}/>

### Subtle Secondary

<Example Component={TypeSubtleSecondaryExample}/>

### Destructive

A destructive button is used to warn the user about a potentially dangerous or destructive action.

<Example Component={TypeDestructiveExample}/>

## Disabled

Disabled buttons indicate that a certain action is not currently available.

<Example Component={DisabledExample}/>

## Toggled

A button can be in a **toggled** state using the `toggled` prop. This is only applicable for subtle variants.

<Example Component={ToggledExample}/>

## Size

Button component has a `size` prop that can be used to change the size of the button.

<Example Component={SizeExample}/>
