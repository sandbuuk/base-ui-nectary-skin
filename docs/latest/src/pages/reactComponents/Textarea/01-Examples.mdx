Shows the resize handle to manually resize the height of textarea.

## Rows

Initial Textarea height defined as number of text lines.

<Example Component={RowsExample}/>

## Min rows

Used with `resizable` prop, to prevent shrinking Textarea height below certain number of text lines.

In this example `minRows` value is set to `3`, so you cannot shrink the height of the Textarea below 3 lines.

<Example Component={MinRowsExample}/>

## Max rows

Used with `resizable` prop, to prevent expanding Textarea height above certain number of text lines.

In this example `maxRows` value is set to `6`, so you cannot expand the height of the Textarea above 6 lines.

<Example Component={MaxRowsExample}/>
