# Tooltip

## Examples

Tooltip shows contextual help about specific target component when user hovers on them.

It's built into [Icon Button](?version=latest&path=/components/iconbutton/examples) and [Color Menu](?version=latest&path=/components/colormenu/examples) by default.

### Orientation

Orientation – where it *points to* from origin.

See API section for all the possible values.
