# Tooltip

## Examples

Tooltip shows contextual help about specific target component when user hovers on them.

It's built into [Icon Button](?path=/components/iconbutton) and [Color Menu](?path=/components/colormenu) by default.

### Orientation

Orientation – where it *points to* from origin.

See API section for all the possible values.
