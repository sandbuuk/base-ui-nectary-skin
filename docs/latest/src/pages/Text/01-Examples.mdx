# Examples

The Nectary text component is a versatile tool for displaying text within your designs. It is a simple and effective way to add clear, concise text to your designs.

## Type

 It offers four different sizes to choose from, allowing you to effectively communicate hierarchy and emphasis within your designs.

<Example Component={TypeExample}/>

## Nested

It is also possible to nest the text component. It allows you to add text within other text or design elements in your projects. This can be useful for creating lists, nested headings, add bold, or other types of hierarchical content.

<Example Component={NestedExample}/>

## Ellipsis

The ellipsis property can be used within the Nectary text component to indicate that there is more content available than what is currently being displayed. It is represented by three dots (...) and is often used to truncate long pieces of text or to show that a user can click or tap to see more. The ellipsis is a useful tool for creating a sense of intrigue and encouraging users to interact with your content.
