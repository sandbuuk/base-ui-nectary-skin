# Examples

A phone input component is a user interface element that allows users to input and validate a phone number. It includes a country code selector. The component may also include validation to ensure the phone number is correctly formatted and of the appropriate length.

## Composition

Phone Input is a composition of [Input](?path=/components/input/examples), [Popover](?path=/components/popover/examples), [Select Button](?path=/components/select-button/examples), [Select Menu](?path=/components/selectmenu/examples) and [Flag](?path=/components/flag/examples).
