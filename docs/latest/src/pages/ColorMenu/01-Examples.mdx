# Color Menu

## Examples

Color Menu is a building block for high-level compositions like [Color Select](?path=/components/colorselect).

A color menu component is a user interface element that allows users to select a specific color from a predefined set of options. Use down/up/right/left arrow keys to navigate and space/enter to select with keyboard.

### Color Set

A color menu component allows users to select colors from a predefined set, and includes the ability to specify color sets. The color menu can be customized and can help users choose cohesive, harmonious color schemes for their designs.

#### Light

The light colors in the color menu are typically used to convey a sense of calm, tranquility, or softness. In this particular implementation, the color menu includes a subset of light colors, which are colors that are generally lighter in tone or saturation.
