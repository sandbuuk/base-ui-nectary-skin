# Title

## Examples

The title component allows to add a title or heading to the user interface. It typically consists of a large, bold font that is used to draw attention and convey the main purpose or topic of the content it is paired with. The title component is an important tool for creating hierarchy and structure within the UI, helping users to quickly understand the content and navigate your user interface.

Whether you are adding a title to a page, a section, or a block of content, the title component is an essential part of Nectary.

### Ellipsis

The ellipsis property can be used within the Nectary text component to indicate that there is more content available than what is currently being displayed. It is represented by three dots (...) and is often used to truncate long pieces of text or to show that a user can click or tap to see more. The ellipsis is a useful tool for creating a sense of intrigue and encouraging users to interact with your content.
