It's a stateless and [controlled](https://reactjs.org/docs/forms.html#controlled-components) component.

## API

### Slots

#### tooltip

`tooltip` slot puts a tooltip into its predefined place, see example above.

#### icon

`icon` slot puts an icon to the left side.

#### left

`left` slot puts arbitrary components to the left side of the input.

#### right

`right` slot puts arbitrary components to the right side of the input.
