# File Status

File Status represents a file state:

* `pending`
* `loading`
* `progress`
* `success`
* `error`

## Table of contents

## Examples

### Pending

File has been picked. Action button should remove file from the list.

<Example Component={PendingExample}/>

### Loading

File is uploading. Action button should cancel the uploading process.

<Example Component={LoadingExample}/>

### Progress

File is uploading. Action button should cancel the uploading process.

<Example Component={ProgressExample}/>

### Success

File has been uploaded. Action button should cancel the uploading process, but since it's an async operation there is a variant with spinner.

<Example Component={SuccessExample}/>

### Error

Something went wrong during file uploading. Action button should either remove file or re-try the failed operation.

<Example Component={ErrorExample}/>
