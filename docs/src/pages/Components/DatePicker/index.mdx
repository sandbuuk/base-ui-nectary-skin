# DatePicker

DatePicker communicates using an [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) (e.g. `YYYY-MM-DD`) date format. Working with timezones is not a part of the component functionality and all internal date manipulations go through consumer's local timezone reported by browser.

## Examples

### Input composition

The component does *not* provide an input field and does *not* do any input masking. It's a modular *picker* and requires you to compose an input + popover + picker on your own, maintaining all the states.

We'll implement a generic masking functionality for the input field  (like `pattern` attribute) later and then maybe wrap it all into `DateInput` complete molecule.
